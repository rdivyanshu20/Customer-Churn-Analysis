# Project 1: Customer Churn Prediction 

##  Overview
This project focuses on predicting which customers are likely to stop using a service based on their historical behavior. By leveraging predictive analytics, this tool helps companies proactively identify at-risk customers, allowing them to implement retention strategies and reduce overall churn.

##  Project Goal
To understand how predictive analytics helps companies retain customers and reduce churn by analyzing past transactions and behavioral data.

##  Concepts & Skills Demonstrated
*   **Data Preprocessing and Cleaning:** Handling missing values, encoding categorical variables, and scaling numerical features.
*   **Exploratory Data Analysis (EDA):** Visualizing data distributions and identifying trends associated with customer attrition.
*   **Feature Selection:** Dropping irrelevant identifiers and selecting the most impactful features for predictive modeling.
*   **Classification Models:** Implementing machine learning algorithms, specifically **Logistic Regression** and **Decision Trees**.
*   **Model Evaluation:** Assessing model performance using key metrics such as **Accuracy** and **ROC-AUC**.

##  Technologies Used
*   Python 3.x
*   Pandas & NumPy (Data Manipulation)
*   Matplotlib & Seaborn (Data Visualization)
*   Scikit-Learn (Machine Learning & Evaluation)
